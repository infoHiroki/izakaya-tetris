# 居酒屋テトリス

飲み食いしてブロックを落とすパーティーゲーム。

## URL
- **本番**: https://izakaya-tetris-game.vercel.app/
- **GitHub**: https://github.com/infoHiroki/izakaya-tetris

## 構成
```
izakaya-tetris/
├── index.html          # アプリ本体（シングルHTML、全機能内包）
├── game-logic.js       # テスト用ロジック抽出モジュール
├── test/
│   └── game-logic.test.js  # vitest テスト (47件)
├── package.json
├── README.md
└── CLAUDE.md
```

## 技術
- シングルHTML（フレームワークなし）
- バニラJS + HTML5 Canvas
- SRS準拠テトリスエンジン（ウォールキック・ロックディレイ対応）
- パーティクルエンジン内蔵（各アクションで発火）
- CSS変数 + グラスモーフィズム + backdrop-filter
- localStorage自動セーブ
- vitest でロジックテスト

## ゲーム設計

### 賞金テーブル（行消しごとの固定報酬・累積制）
| 難易度 | 速度(ms) | 1行 | 2行 | 3行 | テトリス |
|---|---|---|---|---|---|
| 🐢 ゆるゆる | 500 | 500円 | 2,000円 | 4,000円 | 8,000円 |
| 🏃 ふつう | 300 | 1,000円 | 3,000円 | 6,000円 | 12,000円 |
| 🔥 ガチ | 180 | 1,500円 | 5,000円 | 10,000円 | 20,000円 |
| 💀 鬼 | 100 | **0円** | 7,000円 | 15,000円 | 30,000円 |

- 鬼は1行消しでは賞金ゼロ（ハイリスクハイリターン）
- 賞金は行消しごとに加算されていく

### ミニテトリス（盤面生成）
- 15段ボードでプレイ → ゲームオーバー盤面を20段ボードの下15段にコピー（上5段空き）
- 10行消すごとに加速（-40ms、下限80ms）
- ロックディレイ付き（着地後2ティック猶予、移動/回転でリセット）

### 本編ルールバリアント
- v1(シンプル): 1ストック=1ドロー
- v2(おトク): 2ストック=ランダム+自由配置
- v3(ギャンブル): 混合デッキ（ブロック+FREE+MISS）

### 封印（除外）
- 1ストックでブロック種類を次の1ドローだけ封印
- 確認ダイアログ付き
- ドロー後にリセット（デッキからは消えない）

### 操作
- ファミコン風コントローラー（左=十字キー、右=回転ボタン）
- DAS 167ms / ARR 33ms（テトリスガイドライン準拠）
- 要素ベースのタイマー管理（素早い左右切り替えに対応）
- touchcancel対応（指がボタン外にスライドしても停止）
- 全ボタンにハプティクス（移動8ms/回転12ms/ドロップ15ms）
- キーボード対応（矢印キー）

### Undo/編集
- pushHist()で状態スナップショット保存（max 20）
- 食った！/ ドロー / 封印すべてundo可能
- 配置中キャンセル可能
- ↩ボタンをゲーム画面に常時表示
