# 🍺🧩 居酒屋テトリス

酒を飲め！メシを食え！ブロックを落として賞金ゲットだ！

**🎮 プレイ: https://izakaya-tetris-game.vercel.app/**

## 遊び方

### 1. 難易度を選ぶ
| 難易度 | 速度 | 1行 | 2行 | 3行 | テトリス |
|---|---|---|---|---|---|
| 🐢 ゆるゆる | ゆっくり | 500円 | 2,000円 | 4,000円 | 8,000円 |
| 🏃 ふつう | 標準 | 1,000円 | 3,000円 | 6,000円 | 1.2万 |
| 🔥 ガチ | 速い | 1,500円 | 5,000円 | 1万 | 2万 |
| 💀 鬼 | 超速 | **0円** | 7,000円 | 1.5万 | 3万 |

💀 鬼は1行消しでは1円ももらえない！2行以上同時に消せ！

### 2. テトリスで盤面をつくる
- テキトーにテトリスして積み上げろ！
- ゲームオーバーになった盤面が本番のスタート地点
- ロックディレイ付き（着地しても少し滑れる）

### 3. 居酒屋で飲み食いしてブロックを落とす
- 1品食べる/1杯飲む → 「🍺食った！飲んだ！」でストック+1
- ストックで「🎲引く」→ ランダムなブロックが降ってくる
- 制限時間内に操作して配置
- 行を揃えて消す → **賞金が加算されていく**
- **最下段が消えたらクリア！**

### 4. 封印（戦略）
- ストック1個で邪魔なブロックを封印！
- 封印したブロックはもう出てこない → 欲しいブロックの確率爆上がり
- 確認ダイアログ付きなので誤操作も安心

## 操作方法

### スマホ（ファミコン風コントローラー）
- **左 ◀▶ 右** → 移動（長押しで連続移動）
- **▼** → 即落下
- **右の赤ボタン ↻** → 回転
- タッチ操作: タップで回転、左右スワイプで移動、下スワイプで即落下

### PC
- **← →** → 移動
- **↑** → 回転
- **↓** → 即落下

## ルールバリアント

| モード | 内容 |
|---|---|
| ⚡ シンプル | 1品で1回引ける |
| 🔥 おトク | 2品で「ランダム1回 + 好きなの1回」のセット |
| 💀 ギャンブル | デッキにFREEとMISSが混ざってる！運ゲー |

## 便利機能

- **↩ undo** — いつでも1タップで元に戻せる（食った！も戻せる）
- **配置キャンセル** — ブロック配置中に✕ボタンで中断
- **📋 記録** — アイコン・色分けの見やすいアクティビティログ
- **💾 セーブ** — localStorageに自動保存。途中から再開可能
- **パーティクル演出** — 食った/ドロー/行クリア/勝利で光る

## 技術

- シングルHTML（フレームワークなし）
- バニラJS + HTML5 Canvas
- SRS準拠テトリスエンジン（ウォールキック対応）
- パーティクルエンジン内蔵
- CSS変数 + グラスモーフィズム
- localStorage自動セーブ
- スマホ最適化（100dvh、タッチ操作、safe-area対応）
- Vercel自動デプロイ
