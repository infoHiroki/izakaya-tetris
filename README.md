# 🍺🧩 居酒屋テトリス

飲み食いしてブロックを落とすパーティーゲーム。

**🎮 プレイ: https://izakaya-tetris-game.vercel.app/**

## 遊び方

### 1. 難易度を選ぶ（賞金が決まる）
| 難易度 | 速度 | レート | 足切り |
|---|---|---|---|
| 🐢 ゆっくり | 遅い | ×100円 | なし |
| 🏃 ふつう | 標準 | ×300円 | 100pt未満ゼロ |
| 🔥 はやい | 速い | ×1,500円 | 200pt未満ゼロ |
| 💀 鬼 | 超速 | ×8,000円 | 50pt未満ゼロ |

高難易度ほどハイリスクハイリターン。下手だと賞金ゼロ。

### 2. テトリスでゲームオーバーまでプレイ
- スコアが高いほど賞金UP
- だんだん速くなる
- ゲームオーバー時の盤面が本編のスタート

### 3. 居酒屋で飲み食いしてブロックを落とす
- 1品食べる/1杯飲む → 「🍺食った!」でストック+1
- ストックで「🎲ブロックを引く」→ ランダムなブロックが降ってくる
- 操作して配置。制限時間あり
- 行を揃えて消す。**最下段が消えたら勝利!**

### 4. 封印（戦略）
- ストック1個で不要なブロックを永久封印
- 封印したブロックはもう出ない → 欲しいブロックの確率UP

## 操作方法

### スマホ
- **タップ** → 回転
- **左右スワイプ** → 移動
- **下スワイプ** → 即落下
- 画面下のボタンでも操作可

### PC
- **← →** → 移動
- **↑** → 回転
- **↓** → 即落下

## ルールバリアント

| モード | 内容 |
|---|---|
| ⚡ かんたん | 1ストック=1回引いて配置 |
| 🔥 ノーマル | 2ストック=ランダム1回+自由1回 |
| 💀 ハード | 混合デッキ。FREEを引ければ自由配置 |

## 技術

- シングルHTML（フレームワークなし）
- SRS準拠テトリスエンジン（ウォールキック対応）
- localStorage自動セーブ
- スマホ最適化（100dvh、タッチ操作、safe-area対応）
